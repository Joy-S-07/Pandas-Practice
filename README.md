# 🐼 Pandas Practice

A hands-on collection of Pandas exercises and code snippets to strengthen data analysis skills using Python.

---

## 📌 About

This repository contains my personal practice work with the **Pandas** library in Python. It covers core concepts from data loading and cleaning to aggregation and analysis — built as a learning reference and portfolio.

| File / Notebook | Description |

---

## 🧠 Topics Covered

- **DataFrame & Series basics** — creation, structure, data types
- **Data loading** — reading CSV, Excel files
- **Exploring data** — `head()`, `tail()`, `info()`, `describe()`, `shape`
- **Indexing & selection** — `loc`, `iloc`, boolean filtering
- **Data cleaning** — handling missing values with `isnull()`, `fillna()`, `dropna()`
- **Sorting & ranking** — `sort_values()`, `nlargest()`, `nsmallest()`
- **GroupBy & aggregation** — `groupby()`, `agg()`, `value_counts()`
- **String operations** — `str` accessor methods
- **Adding & modifying columns** — computed columns, `apply()`, conditional logic
- **Duplicate detection** — `duplicated()`, `drop_duplicates()`
